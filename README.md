# Backend-Bolierplate

## Description

[Boilerplate](https://github.com/withchandan/backend-boilerplate-v1) backend-boilerplate repository.

## Installation

```bash
# install dependencies
$ npm install
```

## Running the app

```bash
# serverless
$ npm run start:sls

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Linting & Formatting

```bash
# Check & fix
$ npm run lint

# Code format
$ npm run format
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
